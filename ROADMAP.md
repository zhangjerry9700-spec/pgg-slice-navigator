## 待开发功能模块

### Phase 1: 学习体验优化（高优先级）

#### 1.1 连续学习天数追踪
- **功能**：记录并显示真实的连续学习天数
- **细节**：
  - 检测每日首次答题，更新连续天数
  - 断签后从 1 重新开始
  - 首页显示连续天数徽章
- **预估**：30 分钟

#### 1.2 错题专项练习模式
- **功能**：专门针对错题的练习任务
- **细节**：
  - 首页新增「错题复习」任务卡片
  - 从错题本中抽取题目组卷
  - 答对后自动移出错题本
- **预估**：45 分钟

#### 1.3 语法点详情页
- **功能**：查看每个语法主题的详细解释和例题
- **细节**：
  - 点击掌握度网格中的语法点进入
  - 显示该语法点的所有知识卡片
  - 提供「专项练习」入口（只练习该语法点的题目）
- **预估**：60 分钟

### Phase 2: 数据与统计（中优先级）

#### 2.1 学习历史记录
- **功能**：查看过去的学习趋势
- **细节**：
  - 新增「历史」页面
  - 显示每日答题数量折线图
  - 显示正确率趋势
  - 显示各语法点掌握度变化曲线
- **预估**：60 分钟

#### 2.2 数据导入/导出
- **功能**：备份和恢复学习数据
- **细节**：
  - 导出为 JSON 文件
  - 从 JSON 文件导入
  - 防止数据丢失
- **预估**：30 分钟

#### 2.3 题目收藏功能
- **功能**：收藏有价值的题目（不同于错题）
- **细节**：
  - 练习页添加「收藏」按钮
  - 分析页新增「收藏夹」标签
  - 收藏题目可单独练习
- **预估**：45 分钟

### Phase 3: 功能增强（中优先级）

#### 3.1 练习计时功能
- **功能**：记录每题和每次练习的用时
- **细节**：
  - 练习页显示计时器
  - 完成摘要显示平均用时
  - 历史记录显示答题速度趋势
- **预估**：30 分钟

#### 3.2 夜间模式
- **功能**：深色主题切换
- **细节**：
  - 导航栏添加主题切换按钮
  - 使用 CSS 变量实现主题切换
  - 保存用户偏好到 localStorage
- **预估**：45 分钟

#### 3.3 键盘快捷键
- **功能**：支持键盘操作练习
- **细节**：
  - 1-4 选择选项
  - Enter 提交/下一题
  - B 加入错题本
  - H 显示帮助
- **预估**：30 分钟

### Phase 4: 内容扩展（低优先级）

#### 4.1 扩充题库至 200 题
- **功能**：添加更多语法主题
- **新增主题**：
  - 动词时态（现在时、过去时、完成时）
  - 名词变格与性数格
  - 代词（人称代词、指示代词、不定代词）
  - 介词与格/第四格搭配
  - 句序（动词二位、框型结构）
- **预估**：90 分钟

#### 4.2 题目搜索与筛选
- **功能**：按条件查找题目
- **细节**：
  - 按语法点筛选
  - 按难度筛选
  - 按年份筛选
  - 关键字搜索
- **预估**：45 分钟

#### 4.3 成就系统
- **功能**：学习里程碑徽章
- **细节**：
  - 首次答题、连续 7 天、答对 100 题等
  - 首页显示最近获得的成就
  - 分析页显示所有成就
- **预估**：60 分钟

### Phase 5: 技术优化（低优先级）

#### 5.1 PWA 支持
- **功能**：可安装为桌面应用
- **细节**：
  - 添加 manifest.json
  - 添加 Service Worker
  - 支持离线练习
- **预估**：45 分钟

#### 5.2 响应式优化
- **功能**：移动端体验优化
- **细节**：
  - 优化手机端布局
  - 触摸友好的按钮大小
  - 底部导航栏
- **预估**：60 分钟

#### 5.3 性能优化
- **功能**：提升加载速度
- **细节**：
  - 题库分块加载
  - 图片懒加载
  - 虚拟列表（如果有长列表）
- **预估**：45 分钟
